# auto-ml-explorer

A tool for automated machine learning model selection and hyperparameter tuning

## Introduction
This project aims to simplify the process of selecting and tuning machine learning models by providing an automated workflow.
## Features
* Automated model selection from a range of popular algorithms
* Hyperparameter tuning using Bayesian optimization
* Model evaluation and comparison using various metrics
## Usage
To get started, simply install the required dependencies and run the `auto_ml_explorer` script with your dataset as input.
